# Algorithmic Trading Bot - Institutional Order Flow

An automated, quantitative trading bot designed to backtest and execute short-term systematic strategies based on institutional liquidity and market microstructure.

## 🤖 Key Features
- **SMC/ICT Strategy Logic:** Automated identification of institutional order blocks, liquidity pools, and market structure shifts (MSS).
- **Execution Bridge:** Native integration via Python APIs to connect quantitative logic directly with trading execution platforms (**MetaTrader 5 / NinjaTrader 8**).
- **Strict Risk Management:** Built-in calculation modules for dynamic Risk-to-Reward ratios, automated daily drawdown ceilings, and adaptive position sizing.

## 🛠️ Tech Stack
- **Languages & Libraries:** Python, Pandas, NumPy, MetaTrader5 API.
- **Environment:** VS Code, Git.
